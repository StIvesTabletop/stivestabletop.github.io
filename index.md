# What is St. Ives Tabletop?

St Ives Tabletop is a fortnightly tabletop gaming group for the area around St. Ives, Cambridgeshire.

We cover:
* Board games, of which we have hundreds between us!
* Hosting tabletop role-playing games (RPGs).

We play many different board games at club sessions - usually some light games at the start and end, with some mid and heavy weight games in between! You are welcome to bring your own if you are happy to teach, or just turn up and join a game that others have brought.

We are happy to host RPG sessions but we don't run a regular session at club, and most members are board game players, but maybe happy to join one shots. You are welcome to use our [Discord][Discord] server to arrange players for a longer campaign.

# When and where do we meet?

We hold fortnightly sessions on Wednesday evenings at the [St. Ives Corn Exchange](/Directions.html).

Please [contact us][Contact] or join the [Discord][Discord] server if you have any questions!
